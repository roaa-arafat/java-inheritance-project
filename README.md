# Java Inheritance Project

Java OOP inheritance example demonstrating a simple class hierarchy using `Person`, `Employee`, and `Manager` classes.

This project demonstrates basic Object-Oriented Programming (OOP) concepts in Java, including:

- Inheritance
- Encapsulation
- Constructors
- Method Overriding
- Using the `super` keyword

## Classes Overview

### Person
- Attributes: `age`, `name`
- Includes constructors, getters, setters, and an overridden `toString()` method.

### Employee
- Inherits from `Person`
- Adds `id` and `salary`
- Demonstrates constructor chaining using `super`.

### Manager
- Inherits from `Employee`
- Adds `department`
- Extends the functionality of the `Employee` class.

### Test
- Creates objects of `Person`, `Employee`, and `Manager`
- Demonstrates inheritance and object usage.
